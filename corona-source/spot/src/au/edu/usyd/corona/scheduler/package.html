This package contains the implementation of the main scheduler of the system, as well as the implementations of tasks executed in the system.

Package Specification

The main scheduler of the system is found in this package, which coordinate all of the execution of "tasks". Tasks are generated by the query language compiler and distributed throughout the network, to be executed on the {@link au.edu.usyd.corona.scheduler.Scheduler} of each node.

The {@link au.edu.usyd.corona.scheduler.Scheduler} accepts any instance of a {@link au.edu.usyd.corona.scheduler.SchedulableTask} to be added to it, which will be executed as close as possible to its desired execution time.
